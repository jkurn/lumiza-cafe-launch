
# Lumiza Cafe — Single-Page Website

## Design
- **Color palette**: Woody Brown (#3c2e2f), Congo Brown (#613b40), Soya Bean (#6a5e52), with cream/off-white backgrounds for contrast and warm gold accents for CTAs
- **Typography**: Warm serif for headings (Playfair Display), clean sans-serif for body (Inter)
- **Layout**: Single-page scrolling site with sticky navigation, mobile-first

## Sections (top to bottom)

### 1. Navigation Bar
- Sticky top bar with cafe name/logo, smooth-scroll links: Home, Menu, About, Contact
- Mobile hamburger menu
- "Get Directions" button always visible

### 2. Hero Section
- Full-width hero image placeholder (food/interior shot)
- Welcoming tagline: "Your hidden gem in Bondi Junction"
- Opening hours displayed prominently
- Address: 5/28 Spring St, Bondi Junction (under the Holiday Inn)
- Two CTA buttons: **View Menu** and **Get Directions** (links to Google Maps)

### 3. Menu Section
- Clean text-based menu organized by categories: Coffee, Breakfast Classics, Veggie Options, Lunch, Drinks
- Each item shows name, description, and price
- Populated with sample items (Big Breakfast, veggie options, coffee drinks) — easy to edit in code
- Mobile-friendly card/list layout

### 4. About Us Section
- Photo placeholder for Fanny & Shaw
- Authentic paragraph about the owners and their philosophy of great food without the high price tag
- Warm, personal tone

### 5. Contact & Location Section
- Embedded Google Map showing the cafe location
- Address, phone number, opening hours
- Social media icon links (Instagram, Facebook)

### 6. Footer
- Copyright, links, social icons

## SEO & Performance
- Semantic HTML with proper heading hierarchy
- Meta tags optimized for "Cafe in Bondi Junction," "Breakfast near Holiday Inn Bondi," "Best coffee Bondi Junction"
- Fast-loading, minimal dependencies
- Open Graph tags for social sharing

## Content Management
- Menu items and hours stored in a simple data file, making it straightforward to update values without touching layout code
